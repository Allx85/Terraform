# Terraform

## Inledning
Tanken med labben var att labba med tekniker som är "buzzwords" i branschen.

Guiden som följts är [hashicorp guide](https://developer.hashicorp.com/terraform/tutorials/docker-get-started/docker-build)

## Utförande

* Installera Terraform

* Skapa en lätt IaC

* Skapa en `variables.tf` där man skapar variabler för att läsa in i `main.tf`

## Bas kommandon

* `terraform init` - initializing

* `terraform apply` - skapar

* `terraform destroy` - tar bort
